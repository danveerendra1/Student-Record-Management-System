# 🎓 Student Record Management System

A **console-based CRUD (Create, Read, Update, Delete) application** developed in **Java** as part of the **Elevate Labs Java Developer Internship**.

This project demonstrates the implementation of **Object-Oriented Programming (OOP)** principles and Java Collections by providing an interactive menu-driven system to manage student records efficiently.

---

## 📌 Project Overview

The Student Record Management System allows users to manage student information through a command-line interface. It enables users to add, view, search, update, and delete student records while maintaining clean, modular, and reusable code.

---

## ✨ Features

- ➕ Add Student
- 📋 View All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- 🗑️ Delete Student
- 🚫 Duplicate Student ID Validation
- ✅ Marks Validation (0–100)
- 📊 Display Total Students
- 🖥️ Menu-Driven Console Interface
- 📑 Formatted Table Output
- ❌ User-Friendly Error Messages

## ▶️ How to Run

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Open the Project

Open the project in **VS Code** or **IntelliJ IDEA**.

### 3. Navigate to the Source Folder

```bash
cd src
```

### 4. Compile the Project

```bash
javac *.java
```

### 5. Run the Application

```bash
java StudentManagementSystem
```
